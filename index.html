Predicitng the 2023 best picture award

In this project, I embarked on the exciting challenge of predicting the Oscars best picture award, not only one of the most prestigious award ceremoniesm, but also the most coviteded award in the film industry. Using data analytics and machine learning techniques, I aimed to leverage historical data and various film-related factors to forecast the winner accurately.

  • View Data Cleaning Phase
  • View Prediction Phase

Scope

Over 3000's records of unclean and historical data that would eventually be used to predict the winner

Cleaning, Transforming, and Exploring

Making sure no missing data for all relevant features. Transforming datatypes as well as making the datasets normalized. Seeing if there were any insights within the data before building a comprehensive machine learning model

Prediction Model

Utilizing techniques like cross-validation to train my model and recursive feature elimination. As well as classification models like Logistic Regression and Random Forest Classifier
